Smart Review AI - Chrome Extension

This application is designed to run as a Chrome Extension on top of Google Maps.

1. Build the Extension

Run npm run build in your terminal. This will create a dist/ folder containing content.js and `manifest.json`.

2. Load in Chrome

Open chrome://extensions, enable Developer Mode, click Load Unpacked, and select the folder containing manifest.json.

3. Test

Go to any Google Maps place (e.g. a restaurant) and open the Reviews tab. The panel will appear automatically.
